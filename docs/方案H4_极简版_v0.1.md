# 方案 H4 — 极简版向量检索架构 v0.1

- **创建日期**: 2026-04-10
- **状态**: 验证中
- **定位**: 融合 H3 方案演进 + 4 模型对比评测结论, 极简化落地

---

## 0. 核心理念

**两个模型, 搞定一切。**

| 模型 | 用途 | 速度 |
|------|------|------|
| **SSCD ResNet50** | 向量 (搜索排序) | 500/s per GPU |
| **CLIP ViT-L-14** | 标签 (语义过滤) | 350/s per GPU |

不存 CLIP/DINOv2/SigLIP 向量, 不用 VLM, 不用 Grounding DINO。

---

## 1. 架构总览

```
每张图入库 (实时, 120s SLA):
  SSCD 全图        → sscd_vec (512d)    → Milvus
  SSCD 中心60%     → center_vec (512d)  → Milvus (图中图辅助)
  CLIP 零样本分类  → 9 个标签字段       → Milvus
  pHash + dHash    → 128bit 指纹       → Milvus
  
搜索 (80 QPS):
  mode=instance  → SSCD 向量搜索
  mode=semantic  → CLIP 提取标签 → 标签过滤 + SSCD 排序
  mode=exact     → pHash 汉明距离 + SSCD 兜底
  mode=auto      → 标签过滤 + SSCD 排序 (默认)
```

## 4. 搜索策略

### 4.1 mode=instance (实例搜索, 默认)

```
用途: 找原图/裁剪/截图/同款/视角变体
流程: query → SSCD encode → 搜索 sscd_vec + center_vec → 合并 Top K
延迟: ~5ms (P50)
```

### 4.2 mode=semantic (语义搜索)

```
用途: 找"类似的红色T恤" / 跨品牌同类
流程: query → CLIP 提取标签 → 标签过滤 + SSCD 排序
  WHERE category_l2 = T恤 AND color = 红
  ORDER BY SSCD similarity

延迟: ~10ms (P50), CLIP 标签提取 +3ms
```

### 4.3 mode=exact (精确搜索)

```
用途: 找完全相同的图 (原图, 压缩, 水印)
流程: query → pHash → 汉明距离 < 5 → 如无命中则 fallback 到 SSCD
延迟: ~2ms (P50)
```

### 4.4 mode=auto (智能融合, 推荐默认)

```
流程: 
  同时跑 SSCD + 标签过滤 (如有标签)
  合并排序: 0.8 × SSCD_score + 0.2 × tag_match_score
```

## 6. 10 场景验收结果 (实测)

### 6.1 评测数据

| 场景 | 验收标准 | SSCD R@K | 达标? |
|------|---------|---------|------|
| 原图 | Top 1 = 原图 | R@1 = 96.9% | ⚠️ +pHash 可达 100% |
| 压缩/转发 | Top 3 | R@3 = 99.5% | ✅ |
| 轻度裁剪 70% | Top 8 | R@8 = 99.8% | ✅ |
| 轻度形变 | Top 10 | R@10 = 100% | ✅ |
| UI叠加/涂鸦 ≤30% | Top 5 | R@5 = 99.7% | ✅ |
| 重度裁剪 30% | Top 10 | R@10 = 99.3% | ✅ |
| 重度涂鸦 | Top 5 | R@5 = 99.2% | ✅ |
| 分辨率缩放 | Top 10 | R@10 = 100% | ✅ |
| **图中图** | Top 10 | **R@10 = 8.2%** | **❌ 需 L2 增强** |
| **反图中图** | Top 10 | **R@10 = 57.2%** | **❌ 需 L2 增强** |

### 6.2 图中图解决方案

```
实时: 中心裁剪 60% → SSCD → center_vec (额外存储)
      预期提升: 10% → 30-50%

异步 (可选): YOLO 检测 → 精确子图 → SSCD → sub_vec
      预期提升: → 60-80%
```

## 9. 与 H3 方案的差异

| 维度 | H3 v1.1 | **H4 v0.1** |
|------|--------|------------|

| 图中图 | 无 | **中心裁剪 + 异步 YOLO** |
