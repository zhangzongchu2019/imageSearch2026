## 变更详情

### 1. 新增: 以图搜图功能上线

**v5.0 状态**: 需求定稿，未涉及前端交互细节。

**v5.1 新增**:

以图搜图功能正式上线，支持两种输入方式:

| 输入方式 | 说明 |
|---------|------|
| 图片 URL | 输入商品图片链接，系统下载后提取特征搜索 |
| 上传图片 | 用户从本地选择图片上传，支持拍照/相册 |

输出为相似商品图片列表，按匹配优先级排序，展示商品缩略图、相似度分数、品类标签。

---

### 2. 新增: 多图批量搜索

**v5.0 状态**: 场景 3 批量查询定义了 ≤7 张流式返回。

**v5.1 新增**:

增强批量搜索交互体验:

| 功能 | 说明 |
|------|------|
| 多图选择 | 用户可选择多张图片发起批量搜索 |
| 逐张搜索 | 每张图独立执行搜索，结果逐张返回 |
| 缩略图切换 | 搜索结果页顶部展示已选图片缩略图，点击可切换查看对应结果 |

---

### 3. 新增: COS 上传搜索

**v5.0 状态**: 搜索接口仅支持 base64 图片输入。

**v5.1 新增**:

支持通过腾讯云 COS 直传方式上传图片搜索:

| 项目 | 说明 |
|------|------|
| 上传方式 | 腾讯云 COS JavaScript SDK 浏览器端直传 |
| 鉴权方式 | STS (Security Token Service) 临时密钥，有效期 30 分钟 |
| 流程 | 前端申请临时密钥 → COS JS SDK 直传 → 获取 COS URL → 发起搜索 |
| 优势 | 图片不经过应用服务器，减轻带宽压力；支持大图上传 |

---

### 4. 新增: 搜索排序需求 (4 级优先)

**v5.0 状态**: 排序规则为 P0 > P1 > P2，同层级按相似度降序。

**v5.1 新增**:

搜索结果排序细化为 4 级优先级:

| 优先级 | 匹配类型 | 排序说明 |
|--------|---------|---------|
| **第 1 级** | 原图相似 | 相似度 > 0.90，与查询图高度一致 |
| **第 2 级** | 型号一致 | 同款商品，不同角度/颜色 |
| **第 3 级** | 系列相同 | 同系列产品，设计语言一致 |
| **第 4 级** | 颜色材质 | 颜色、材质等属性相近 |

排序规则: 第 1 级永远排最前，依次降级。同级别内按综合相似度降序。

---

### 5. 新增: 品类展示

**v5.0 状态**: 搜索结果包含 category 字段，未定义展示格式。

**v5.1 新增**:

搜索结果中展示一级/二级品类名称，格式为层级路径:

| 示例 | 一级品类 | 二级品类 | 展示 |
|------|---------|---------|------|
| 单肩包 | 箱包类 | 单肩包 | 箱包/单肩包 |
| 短袖T恤 | 服装类 | 短袖T恤 | 服装/短袖T恤 |
| 运动鞋 | 鞋帽类 | 运动鞋 | 鞋帽/运动鞋 |

品类名称来源于 L1 (18 类) + L2 (84 类) 品类体系，由特征提取阶段预测。

---

### 6. 新增: 相似度阈值

**v5.0 状态**: 未定义相似度截断阈值，所有 Top-K 结果均返回。

**v5.1 新增**:

引入相似度阈值过滤，不相关结果不展示:

| 项目 | 说明 |
|------|------|
| 阈值 | raw_score < 0.80 的结果不显示 |
| 目的 | 过滤低相关度结果，避免展示明显不相关的商品 |
| 影响 | 实际返回结果数可能少于请求的 top_k |

---

### 7. 变更: 数据规模 20M → 116M

**v5.0 状态**: 测试环境数据规模约 2000 万 (单分区)。

**v5.1 变更**:

| 指标 | v5.0 | v5.1 | 说明 |
|------|------|------|------|
| 已入库数据总量 | ~20M | ~116M | 多 Collection 迁移完成后的实际数据量 |
| Collection 架构 | 单 Collection | 多 Collection (按月+等级分区) | img_YYYYMM_vip / img_YYYYMM_svip / img_999999 |
| 分区数据分布 | 单一分区 | VIP 热区 + SVIP 热区 + 常青池 + 冷区 | 独立 Collection 隔离 |

## 变更影响评估

| 影响范围 | 说明 |
|---------|------|
| 前端 | 需适配 COS 直传、多图切换、品类展示、阈值截断 |
| 搜索服务 | 需实现 4 级排序、相似度阈值过滤、品类名称映射 |
| 推理服务 | 无变更 (品类预测已支持 L1+L2) |
| 数据层 | 多 Collection 架构已就位，数据规模已扩展 |
