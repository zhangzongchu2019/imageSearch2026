## 变更详情

### 1. 新增: 品类加权搜索策略

**v2.0 状态**: 融合排序公式为固定权重 (global 0.40 + sub 0.30 + tag 0.20 + cat 0.10)，品类匹配为布尔值。

**v2.1 新增**:

品类匹配从布尔值细化为分级加权:

| 品类匹配层级 | 加权分值 | 说明 |
|-------------|---------|------|
| 同 L1 品类 | +0.05 | 查询图与结果图属于相同一级品类 (如都是"箱包类") |
| 同 L2 品类 | +0.03 | 在 L1 匹配基础上，二级品类也一致 (如都是"单肩包") |

加权作用于最终排序分数:

```
final_score = raw_similarity_score + l1_bonus + l2_bonus

示例:
  raw_score = 0.85, 同 L1 + 同 L2 → final = 0.85 + 0.05 + 0.03 = 0.93
  raw_score = 0.85, 仅同 L1       → final = 0.85 + 0.05 = 0.90
  raw_score = 0.85, 品类不同       → final = 0.85
```

### 2. 新增: 两路搜索合并策略

**v2.0 状态**: 检索管线描述了多阶段流程，未明确品类过滤路径的合并方式。

**v2.1 新增**:

搜索执行两路并行，结果合并去重:

| 搜索路径 | 策略 | 返回数量 |
|---------|------|---------|
| **全局搜索** | 不限品类，全局 ANN 搜索 | Top-20 |
| **品类过滤搜索** | 限定与查询图同 L1 品类，ANN 搜索 | Top-20 |

合并规则:
1. 两路结果合并 (最多 40 条)
2. 按 image_id 去重 (保留分数较高者)
3. 应用品类加权 (+0.05/+0.03)
4. 按 final_score 降序排序
5. 应用相似度阈值过滤 (≥0.80)
6. 返回最终 Top-K

两路合并的目的: 全局搜索保证召回多样性，品类过滤搜索保证同品类结果的深度覆盖。

### 3. 新增: 相似度阈值

**v2.0 状态**: P0/P1/P2 匹配层级定义了典型相似度范围，但无硬截断。

**v2.1 新增**:

| 项目 | 说明 |
|------|------|
| 阈值条件 | raw_score >= 0.80 |
| 过滤时机 | 品类加权之前，基于原始相似度分数判断 |
| 过滤行为 | raw_score < 0.80 的结果不返回给前端 |
| 业务含义 | 低于 0.80 的结果通常为不同品类或不同风格，展示会降低用户信任 |

与匹配层级的关系:

| 匹配层级 | 原定义 | 阈值后 |
|---------|--------|--------|
| P0 原图匹配 | > 0.90 | 保留 (> 0.90) |
| P1 款式匹配 | 0.70~0.90 | 仅保留 0.80~0.90 段 |
| P2 品类风格 | 0.50~0.70 | **全部过滤** |

> 注: P2 结果在 v2.1 中被阈值过滤，不再展示。后续可根据业务反馈调整阈值。

### 4. 新增: L2 品类显示 (84 个二级品类)

**v2.0 状态**: 品类体系仅定义了 L1 大类 (18 类)，L2/L3 标注为 0。

**v2.1 新增**:

新增 84 个二级品类映射，搜索结果中展示 L1/L2 品类路径:

| L1 品类 | L2 品类数量 | 示例 |
|---------|-----------|------|
| 服装类 (1001) | 12 | 短袖T恤、长袖衬衫、连衣裙、牛仔裤、西装外套... |
| 箱包类 (1002) | 8 | 单肩包、双肩包、手提包、钱包、行李箱... |
| 鞋帽类 (1003) | 10 | 运动鞋、高跟鞋、靴子、凉鞋、帽子... |
| 珠宝首饰类 (1004) | 8 | 项链、手链、戒指、耳环、胸针... |
| 其他 14 个 L1 | 46 | 各品类细分 |
| **合计** | **84** | |

L2 品类由 FashionSigLIP (时尚类) 和 ViT-L-14 零样本 (非时尚类) 推理得出。

展示格式: `{L1名称}/{L2名称}`，如 "箱包/单肩包"、"服装/短袖T恤"。

### 5. 新增: 搜索精度基线

**v2.0 状态**: 未建立搜索精度基线数据。

**v2.1 新增**:

基于全链路测试 (Top-8, 查询图: 短袖T恤) 建立的精度基线:

| 指标 | 结果 | 说明 |
|------|------|------|
| L1 品类准确率 | **100%** | Top-8 结果全部为"服装类" |
| Top-5 精准率 | **40%** | Top-5 中 2 条为短袖T恤 (精确匹配) |
| Top-8 品类一致率 | 100% | 所有返回结果品类正确 |

> 基线说明: 当前精准率受限于库内数据多样性和模型细粒度区分能力。品类级别准确率优秀，细粒度 (同款/同型号) 精准率有提升空间。

---

### 6. 新增: 待优化项

**v2.0 状态**: 无待优化清单。

**v2.1 新增**:

| 优化项 | 说明 | 优先级 |
|--------|------|--------|
| 属性过滤 | 支持按属性维度过滤: 短袖/长袖、颜色、材质等 | P1 |
| 标签匹配加权 | 查询图标签与结果图标签重合时给予额外加权 | P2 |

属性过滤示例:
```
用户搜索"短袖T恤"图片:
  → 当前: 返回所有服装类结果 (含长袖、外套等)
  → 优化后: 可勾选"短袖"过滤，仅返回短袖类商品
```

标签匹配加权示例:
```
查询图标签: [棉质, 简约风, 白, 夏季]
结果图标签: [棉质, 简约风, 白, 日常]
重合标签: 3/4 → 给予额外加权 (具体权重待定)
```
