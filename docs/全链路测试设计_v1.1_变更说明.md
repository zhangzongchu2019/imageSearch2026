## 一、Milvus 分布式集群测试

### 1.1 集群配置
- 14 组件: 1 Proxy + 4 Coord + 2 DataNode + 3 QueryNode(128GB) + 4 IndexNode
- MQ: Kafka (kafka:9092 PLAINTEXT), etcd 3 节点, MinIO
- 数据: 116M 向量, 5 Collection

### 1.2 性能测试结果

| 指标 | standalone | 分布式 | 提升 |
|------|-----------|--------|------|
| P50 | 4ms | 6ms | - |
| P95 | 14ms | 8ms | 1.8x |
| P99 | 27ms | 10ms | 2.7x |
| 标量过滤 | 374ms | 61ms | 6x |

### 1.3 迁移测试
- standalone → distributed: 零停机, 数据自动识别
- 关键: MQ_TYPE=kafka 必须显式设置 (默认 Pulsar)
- Kafka listener: 用 kafka:9092 (PLAINTEXT), 不是 localhost:29092
